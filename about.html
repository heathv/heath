About me

Designer. Illustrator. All around goober.

 

This process involves breaking each 3 minute video into twelve, 15-second clips. These shorter clips are easier for users to process in large numbers, and allow for a wider array of subject matter to be analyzed in a short period. Each clip is viewed five times, and if the majority agree on what's in the image it is pulled from the set and retired.
